> This repository documents my hands-on study of core Computer Science concepts and Java programming fundamentals. The focus is on building a solid foundation in algorithmic thinking and translating that logic into clean, working code.

---

## 📚 Topics Covered

### 1. Fundamentals
- What algorithms are and why they matter
- The development cycle: Analysis → Algorithm → Coding → Testing → Documentation → Maintenance

### 2. Data & Operators
- **Variables & Constants** — name, type, value, memory address
- **Primitive Types** — integers, floats, `char`, booleans
- **Operators**
  - Arithmetic: `+` `-` `*` `/` `%`
  - Increment / Decrement: `++` `--`
  - Relational: `==` `!=` `>` `<` `>=` `<=`
  - Logical: `&&` (AND) `||` (OR) `!` (NOT)

### 3. Control Flow
- **Conditionals:** `if...else`, nested conditions, `switch`
- **Loops:**
  - `while` — condition checked at the start
  - `for` — best when number of iterations is known
  - `do...while` — always runs at least once

### 4. Input & Output
- Reading user input with the `Scanner` class
- Printing output with `System.out.println()` and escape sequences (`\n`, `\t`)
