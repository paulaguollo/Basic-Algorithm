## How to run

Open in **IntelliJ IDEA** — the project config is already included.

```bash
# Clone the repo
git clone https://github.com/paulaguollo/Basic-Algorithm.git
```

Then: `File → Open` → select the `Algoritmia` folder → run any `ExXX.java` directly.
